✔️ 1. Acesse sua pasta do projeto
-> cd /mnt/c/CaminhoParaProjeto/ProjetoEngenharia-SoftwareEducacionalParaAnaliseESolucaoDeProblemasEmEngenhariaDePoco/src-v2

✔️2. Verifique se você tem um arquivo .pro lá:
-> ls

✔️ 3. Gere os arquivos de compilação com qmake

-> qmake

Se quiser forçar o modo Release:
-> qmake CONFIG+=release


✔️ 4. Compile com make

-> make -j$(nproc)

✔️ 5. Rode o executável
-> ./Simulador-Poco
